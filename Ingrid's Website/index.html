Introduction to a new colorful and full of possibilities world


When you start a tech course you will either be fascinated towards the front or back end, based on your own experiences and preferences. It is for sure important do be able to become a full-stack programmer with multiples talents, but it is hard not to fall for web design.

This Front End Web Development includes not one or two, but three coding languages: HTML, CSS and JAVASCRIPT. Each one is responsible for one part of the construction of a website: HTML is the overall structure, CSS takes control of the design and JS does some kind of magic, interacting with the user through its functionalities.

On their respective pages they will be discussed in more depth but to do a simple exemplification, the following video by Danielle Thé with no more than 4 minutes explains in a easy straight-forward way.

A webpage would look and feel incomplete if some of these three elements were not present. Below a table containing their main purposes:

    HTML Main functionalities
  • Role: The Builder.
  • Responsible for the main structure of the page.
  • Does inline corrections, alterations.
  • Has tags and content, which forms an element.
  • Difers on each web page.
    CSS Main functionalities
  • Role: The Artist.
  • Responsible for the design and graphic elements of the page.
  • Follows the rule "selector{property:value}".
  • Can be input inline, internally (through the tag <"style">) or externally.
  • It's written on a .css sheet and its applicable for the whole website and its pages.
    JS Main functionalities
  • Role: The Wizard.
  • Responsible for the responsiveness and interacting with the user.
  • Can be found with the tag <"script">.
  • Can collect data that later is stored in a database.
  • Four verbs: Acess, Modify, Program, React.
