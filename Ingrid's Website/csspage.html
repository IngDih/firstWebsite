CSS: the Artist

CSS stands for Cascading Style Sheetsand it is a mechanism to add multiple style elements, such as colors, font sizes, padding, letter or paragraph spacing and etc. to a webpage. As mentioned on the index the ways it can be imput are 3:

  1. Inline input: in this type of input, the CSS is written on a html tag, symbolized with the word "style". An example of that is the CSS present in this opening tag:

  2. Internal input: is represented with the style tag and it stands inside the head tag.

  3. External input: this is how this webpage CSS was structured. A stylesheet is stored in another folder and then imported on the head tag of the website. As shown:

    It is preferable to set the stylesheet externally because it is way more practical to import it with one line of code, than include the styling on every single page. It ocupies a bunch of space and make it harder to organize the webpage coding in general.
