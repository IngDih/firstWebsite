Javascript: the Wizard

JavaScript is a high-level programming language (closest to English than machine language) that utilizes the curly bracket as syntax "{ }". It lies alongside HTML and CSS and has the role to interact and study the user behaviour. It can collect data (that will be stored in a data center)

On a Webpage it can be noticed as the action buttons, for example when you run widgets, open PDFs or enter a pin on a website. As a Javascript example, the following code will be demonstrated and transcripted:



The Javascript code responsible for the responsiveness of this button is the following:
<*body> <*script> function myFunction() {
var x = document.getElementById("pinkborder");
if (x.style.display === "none") {
x.style.display = "block";
} else {
x.style.display = "none";
}
}
<*/script><*/body>

And the HTML part is this:
<*button onclick="myFunction()">Show Stars<*/button>


Another example is a print button: : That automatically open the print function of the browser
<*button onclick="window.print()">Print this page<*/button>

Although logically Javascript is the hardest element in Web Programming to learn, because of all the logic and sometimes Maths involved, it is extremelly important to master this area and incorporate its elements to the website, otherwise the user interaction will be low or non existent.
