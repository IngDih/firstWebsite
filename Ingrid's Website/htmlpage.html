HTML: the Builder


HTML stands for HyperText Markup Language and is the standard markup language used in browsers. HTML is in its version number 5, and is known to be used widely by many powerful and influent companies since its release in January of 2008.

What is it about?

HTML consists of many different tags that alter the appearance of text, create tables, links and add images, and by doing so, perfect the structure so the design can lie behind its elements. At first glance HTML and CSS might be a bit difficult to define and separate from each other, but observing the structure of a HTML element might help a beginner to distinguish how the items should look like to be considered a HTML element.


To exemplify, some examples of tags are displayed below:

Tag (Should be put in between "<>") What it does
b, i, u, s Alters text, respectively: bold, italics, underlined, striked
button Creates a button:
img src="photo.jpg" Adds an image to the document.
p Makes the text inside of it a paragraph.
br Breaks a line (can be used as padding in text).
a href="link"> Text cc for the link /a Adds an image to the document.
table Creates a table of content, just like this very table.
ol, ul resp. Ordered list and unordered list. With the "li" (list item) tag creates lists that are marked with numbers or bullet points.
